## 介绍

Springboot-cli开发脚手架，各种常用框架使用案例模板，完善的文档并附带搭建过程讲解，致力于让开发者快速搭建基础环境并让应用跑起来。

- 代码优雅整洁
- 遵守阿里编程规范，高质量、高效率
- 统一的注释方法和风格，提高可读性
- 持续更新维护
