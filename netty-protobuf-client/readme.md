# 基于netty的 protobuf客户端

## 环境



 | #    | 环境 | 版本           | 说明            |
 | ---- | ---- | -------------- | --------------- |
 | 1    | JDK  | openJdk 11.0.8 | 建议JDK11及以上 |



## 项目结构

```
 ├──channel          管道
 ├──config           服务配置
 ├──controller       http测试api
 ├──handler          消息处理器
 ├──protocol         协议文件
 ├──server           服务配置
 ├──utils            工具包
 ├──NettyClientApplication.java   主启动类
```
